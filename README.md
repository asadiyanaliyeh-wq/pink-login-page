💗 پروژه Pink Login Page 🌸






💖 معرفی پروژه

پروژه Pink Login Page یک فرم لاگین مدرن و مینیمال با رنگ‌های صورتی و طراحی چشم‌نواز است.
هدف از این پروژه تمرین طراحی رابط کاربری (UI) و یادگیری گام‌به‌گام استفاده از Git و GitHub در یک پروژه واقعی است.

🎀 ویژگی‌ها

✨ طراحی واکنش‌گرا (Responsive)
🌸 استفاده از گرادیان صورتی ملایم
💫 افکت‌های hover و انیمیشن‌های نرم
🔒 اعتبارسنجی ساده سمت کاربر (JavaScript)
🩷 ساختار تمیز و قابل توسعه

🧠 مهارت‌هایی که در این پروژه تمرین می‌کنی

HTML5 و ساختاردهی معنایی

CSS3 (Flexbox, animation, gradient)

JavaScript برای اعتبارسنجی

مدیریت نسخه با Git

آپلود پروژه در GitHub

🪄 نحوه اجرا (Run Project)

برای دیدن پروژه در مرورگر:

git clone https://github.com/asadiyanaliyeh-wq/pink-login-page.git
cd pink-login-page


سپس فایل index.html را با مرورگر باز کن 🌸
